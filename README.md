# Salt & Stone AU — Demo E-Commerce (Advanced)

Demo storefront for **Salt & Stone AU**, a natural body care brand. Built with Next.js 15, Tailwind CSS 4, and Shopify Storefront API.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS 4
- **E-commerce:** Shopify Storefront API
- **Payments:** Stripe
- **Auth:** Supabase
- **Deployment:** Vercel
